# Guide

## Version requirements

- NodeJS >= 16.15.0

    sude apt install nodejs

- npm >= 9.19.2

    sudo apt install npm


## Installations

- Angular
```
npm install -g @angular/cli@16.2.11
```

- Flex Layout
```
npm i @angular/flex-layout
```

- Cookie Service
```
npm i ngx-cookie-service@16.1.0
```

- JWT Decode
```
npm i jwt-decode
```
- Circle progress
```
npm install --save js-circle-progress
```

## Project launch
```
ng serve
```
